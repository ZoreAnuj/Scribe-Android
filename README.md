# Scribe-Android

Scribe-Android is an Android application designed to aid language learning through specialized keyboards. It explores how custom input methods can provide in-context translations, definitions, and corrections to help users practice a new language directly within any app they use.

## Key Features
*   Custom keyboards for multiple languages with integrated learning aids.
*   Inline translations and word definitions accessible while typing.
*   A clean, intuitive UI built with modern Android development practices.
*   Open-source foundation, allowing for community extension and customization.

## Tech Stack
*   Kotlin
*   Jetpack Compose
*   Android SDK

## Getting Started
1.  Clone the repository: `git clone https://github.com/zoreanuj/Scribe-Android.git`
2.  Open the project in Android Studio.
3.  Build and run the app on an emulator or connected device.
