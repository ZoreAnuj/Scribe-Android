# Scribe-Android

Scribe-Android is a personal exploration into language learning tools, inspired by the original Scribe project. It provides custom Android keyboards designed to help learners input text in their target language more intuitively. This project exists to experiment with mobile-first, in-context learning aids that reduce friction during practice.

## Key Features
*   Custom keyboards for various languages with learner-friendly layouts.
*   Integrated dictionary and translation suggestions accessible during typing.
*   A clean, focused UI that minimizes distraction while practicing.
*   Local data storage for user preferences and learned vocabulary.

## Tech Stack
*   Kotlin
*   Android SDK
*   SQLite (via Room)

## Getting Started
1.  Clone the repository: `git clone https://github.com/zoreanuj/Scribe-Android.git`
2.  Open the project in Android Studio.
3.  Build and run the app on an emulator or connected device.
